## 🚀 Features

- 🎵 **Music Streaming:** Upload and stream MP3 files with ease
- 📝 **LRC Lyrics Sync:** Real-time line-by-line lyrics highlighting
- 🖼️ **Cover Art Support:** Automatic extraction from MP3 files
- 📀 **Album Management:** Create and organize albums with cover art
- 👥 **Collaborative Rooms:** Real-time listening rooms with friends
- 🔄 **Playback Sync:** Everyone hears the same moment
- 📱 **Telegram Bot Integration:** Login and control from Telegram
- 🎨 **Premium UI:** Modern, responsive, and dark theme
- 🔐 **Admin Panel:** Full control over tracks, albums, and users
- 🐳 **Docker Support:** Easy deployment with Docker
